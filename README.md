# Skinome × Holland — Projektmapp

Country Manager-projekt för Skinomes lansering i Nederländerna.

---

## Filer i den här mappen

| Fil | Beskrivning |
|---|---|
| `skinome-holland-strategi.html` | **Huvuddokumentet** — komplett strategi-presentation för GitHub Pages och marketing-teamet. Öppna i webbläsare. |
| `BRIEFING-ÅTERUPPTA.md` | **Starta här vid ny diskussion** — komplett kontext, nuläge, alla nyckelinsikter samlade. Klistra in i ny chat. |
| `research-raw-data.md` | Rådata från marknadsresearch — priser, influencers, Meta-benchmarks, Bol.com, kylkedja. |
| `README.md` | Den här filen. |

---

## Hur man återupptar diskussionen

1. Öppna `BRIEFING-ÅTERUPPTA.md`
2. Klistra in innehållet i en ny Claude-chat med instruktionen: *"Här är kontexten från vår förra diskussion om Skinome Holland. Du är Country Manager. Vi arbetar vidare på [specifik punkt]."*
3. Använd `research-raw-data.md` om du behöver specifika siffror eller källdata.
